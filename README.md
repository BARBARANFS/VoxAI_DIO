# VoxAI_DIO

Assistente de voz desenvolvido como projeto da Digital Innovation One (DIO), integrando reconhecimento de fala e geração de linguagem natural.  
Implementação realizada com **Hugging Face Transformers** em execução local.

---

## 🎯 Objetivo
Criar um assistente de voz capaz de transcrever áudio em texto e responder de forma inteligente, utilizando modelos de linguagem e reconhecimento de fala.

---

## 🚀 Tecnologias utilizadas
- Python  
- Hugging Face Transformers  
- Torch  
- Whisper (Speech-to-Text)  
- Modelos de linguagem estilo ChatGPT  

---

## ⚙️ Funcionalidades
- Conversão de voz em texto (Speech-to-Text).  
- Geração de respostas inteligentes em linguagem natural.  
- Execução local com modelos da Hugging Face.  

---

## 📌 Observações
Este projeto foi inspirado no desafio da DIO (Bootcamp Bradesco GenAI & Dados) para criar um assistente de voz com Whisper e ChatGPT, mas a implementação foi adaptada para rodar com **Transformers localmente**.
